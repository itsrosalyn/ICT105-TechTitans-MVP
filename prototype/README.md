# Prototype Folder

This folder contains the first working prototype of AttendQR - Attendance check system with QR code for Lab 10.

The prototype demonstrates the core functionality of the QR Code Attendance System with GPS Verification. It includes student and lecturer login, QR code generation, QR code scanning, simulated GPS verification, attendance confirmation, attendance history viewing, and lecturer attendance monitoring features. The prototype is implemented as a web-based application using HTML, CSS, JavaScript, Firebase Authentication, and Firestore.
