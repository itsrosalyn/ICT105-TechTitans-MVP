# Wireframe Usability Review Checklist

| Check Item                                                                                                                        | Yes/No | Evidence or Screenshot |
| --------------------------------------------------------------------------------------------------------------------------------- | ------ | ---------------------- |
| The homepage clearly explains the attendance system and provides Student and Lecturer login options.                              |        |                        |
| The student check-in workflow (Login → QR Scan → GPS Verification → Confirmation) can be followed from start to finish.           |        |                        |
| The QR Check-in screen includes QR scanning, GPS verification, and appropriate success/error messages.                            |        |                        |
| The Attendance History screen includes attendance records, attendance percentage, and search/filter controls.                     |        |                        |
| The Attendance Detail screen displays complete attendance information, including date, time, status, and GPS verification result. |        |                        |
| The Lecturer QR Generation screen allows lecturers to create attendance sessions and generate dynamic QR codes.                   |        |                        |
| The Lecturer Dashboard displays meaningful attendance summaries, live attendance records, and attendance statistics.              |        |                        |
| Navigation is consistent across all Student and Lecturer screens.                                                                 |        |                        |
| The wireframes use realistic sample data (student names, course codes, dates, attendance status, percentages, etc.).              |        |                        |
| Every wireframe maps to the project's Functional Requirements (FRs) and MVP features.                                             |        |                        |
