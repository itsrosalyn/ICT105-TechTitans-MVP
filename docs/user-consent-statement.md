# User Consent Statement

This prototype is created for the **ICT105 Fundamental Technology Entrepreneurship** course as part of the **QR Code Attendance System with GPS Verification** project. Participants are invited to test the prototype and provide feedback on its usability and functionality. The information collected will be used only for improving the prototype and completing course-related analysis and documentation. Users should not provide sensitive personal information. Participation is voluntary, and participants may stop the test at any time.

## Data Collected

- Name/nickname or anonymous ID
- User role (Student or Lecturer)
- Feedback score
- Comments about usability
- Task completion result

## Data Use

The collected data will be used only for:

- Evaluating the usability of the prototype
- Improving the QR Code Attendance System with GPS Verification prototype
- Supporting customer validation and analytics
- Preparing course reports and final project documentation for ICT105

All collected information will be used for educational purposes only and will not be shared outside the course without participant permission.
