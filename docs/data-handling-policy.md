# Data Handling Policy

| Section                   | Description                                                                                                                                                                                                                                                            |
| ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Data Collection**       | The prototype collects only the information necessary for attendance management, including Student ID, student name, course/class, attendance date and time, QR code session ID, selected location verification (Near/Far or GPS verification), and attendance status. |
| **Data Storage**          | Attendance data is stored in **Firebase Firestore**. Firebase is used to store and retrieve attendance records, QR code session information, and sample user data for the prototype.                                                                                   |
| **Data Access**           | Students can view only their own attendance history. Lecturers can create attendance sessions, generate QR codes, and view attendance records and dashboard summaries. Access is restricted according to the user's role.                                              |
| **Data Minimization**     | The prototype collects only the information required for attendance tracking. It does not collect unnecessary personal information such as phone numbers, home addresses, or other unrelated personal data.                                                            |
| **Responsible Data Rule** | The prototype stores only the minimum data required for attendance management. Sample or test accounts are used during development where appropriate, and data is accessed only by authorized users. Sensitive information is minimized to protect user privacy.       |
